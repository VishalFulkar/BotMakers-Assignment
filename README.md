# Botmakers Assignment


## Tech Stack
- **Core**: React 19, JavaScript (ES6+), HTML5
- **Build Tool**: Vite 8
- **Styling**: Tailwind CSS v4 (using `@tailwindcss/vite` plugin)
- **Icons**: Custom local SVG assets, inline SVGs

## Responsiveness
- Designed with mobile-first break points (`sm`, `md`, `lg`, `xl`).
- Grid-based layouts automatically shift columns from mobile vertical stacks to multi-column desktop grids.
- Features dual timeline modes (horizontal layout for desktops, vertical layout for mobile viewports).

## Animations
- **Scale Transitions**: Smooth, micro-interactive hover transitions on cards, icons, and buttons.
- **Ambient Glows**: Pulse and glow hover filters highlighting active states and timeline nodes.
